# Translation Project — Brain Directory

**Project:** Dolnytsky Typikon Scientific English Translation  
**Source:** 2010 Lviv Reprint of the 1899 Typikon by Fr. Isydor Dolnytsky  
**Last Updated:** 2026-05-11  

---

## Purpose

This `_brain/` directory is the **single source of truth** for any AI agent or human contributor resuming work on this project. It consolidates all accumulated institutional knowledge from 25+ conversations, distilled into actionable reference documents.

## Directory Structure

```
_brain/
├── README.md                           ← You are here
├── PROJECT_CONTEXT.md                  ← Master project overview & current state
├── SYSTEM_INSTRUCTIONS.md              ← Canonical translation rules (immutable)
├── TERMINOLOGY_GLOSSARY.md             ← Standardized terminology with verdicts
├── VISUAL_AUDIT_PROTOCOL.md            ← Audit methodology & progress tracker
├── DECISION_LOG.md                     ← All key editorial & technical decisions
├── CONVERSATION_INDEX.md               ← Index of all past conversations by topic
├── FILE_MAP.md                         ← Complete file inventory with descriptions
├── PIPELINE_HISTORY.md                 ← History of all processing scripts & phases
└── OPEN_WORK.md                        ← Prioritized remaining tasks
```

## How To Use

1. **Starting a new session?** Read `PROJECT_CONTEXT.md` first.
2. **Resuming the visual audit?** Read `VISUAL_AUDIT_PROTOCOL.md`.
3. **Unsure about a term?** Check `TERMINOLOGY_GLOSSARY.md`.
4. **Need to understand a past decision?** Check `DECISION_LOG.md`.
5. **Looking for a specific file?** Check `FILE_MAP.md`.
